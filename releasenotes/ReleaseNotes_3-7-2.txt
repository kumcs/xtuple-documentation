Release Notes
xTuple ERP
PostBooks, Standard, and Manufacturing Editions
Version 3.7.2
June 17, 2011
==================================

These are the release notes for the 3.7.2 Patch Release. Thanks to 
all who contributed to make this release possible.

==================================

The following features and bug fixes have been added to the
applications since the 3.7.1 release. Additional detail for
each item listed below may be found on our community
website (www.xtuple.org). Simply go to the Issue Tracker and
select the "Changelog" option.

New Features:

* DateCluster supports Site Calendar enforcement (enabled via scripting)

Bug Fixes:

* Setting warehouse work week does not work correctly
* Repositioning of Planned order screen
* Broken Characteristics drop down
* Lock up on two SN transactions
* Accounting assignments won't save on setup
* Can't deactivate item sites
* Not able to issue material as batch to only pick-list items
* Sales Order line item - navigation issue when using different UOMs
* Dropship UOM conversion when SO has different selling UOM
